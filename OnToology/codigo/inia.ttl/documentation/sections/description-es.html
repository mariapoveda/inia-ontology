Ontología datasets INIA: Descripción volver a índice

El objetivo principal de la ontología es representar los metadatos de los dataset almacenados en el portal de datos abiertos de INIA, teniendo en cuenta los metadatos característicos de este tipo de conjuntos de datos. Por ejemplo, los metadatos pueden incluir información sobre el ámbito taxonómico sobre el que se recogen datos en el conjunto de datos o el software con el que los datos deben ser procesados. Sin embargo, un gran número de metadatos de estos conjuntos de datos son comunes a otros catálogos existentes y por lo tanto ya existen ontologías que cubran dichos metadatos. Tras una búsqueda se observó que la ontología dataid es actualmente la más completa para representar metadatos de catálogos y conjuntos de datos reutilizando a su vez ontologías de dominio para temas específicos como lenguajes, licencias, etc. Para completar la representación en rdf de los metadatos necesarios otras ontologías como la Organization ontology o FOAF han sido reutilizadas.

Además, sobre este concpeto inia:Dataset se definen los axiomas de cardinalidad específicos de este caso de uso.
